# Diana Quickstart — 第一次打開的順序

> 目標：10 分鐘內有第一筆真實收入進到系統，不用看完所有文件。

## 1. 啟動

```bash
cd diana-tax
./start.command          # Mac
start.bat                # Windows
```

瀏覽器自動打開 [http://localhost:8501](http://localhost:8501)。

## 2. 想先隨便點看看？跑 demo 資料

```bash
python scripts/seed_demo.py
```

會清空目前的 `data/diana.db` 然後填入 6 筆假收入 + 4 個案主 + 3 張扣繳憑單。
每頁都能看到有資料的樣子。不想要的時候砍掉 `data/diana.db` 即可。

## 3. 清掉 demo，開始用真的

刪除 `data/diana.db` → 重開 streamlit，會自動建立空 DB。

## 4. 第一次的填寫順序

### 第一步：👤 個人設定
- 婚姻狀態、扶養人數 → 影響免稅額
- **職業類別**：最重要！Diana 你若主要寫書稿 / 做講師 → 選「著作人」。
- 健保：若工會加保，選 `union`（影響二代健保提醒）。

### 第二步：➕ 新增收入（試一筆）
- 日期、金額、所得類別（不確定先選 9B_other，之後再改）
- 右側有智能提示：業主會扣多少稅、單筆是否該扣二代健保

### 第三步：📋 收入明細
- 用 Excel-like 表格檢視所有收入
- 可排序、篩選、匯出 CSV

### 第四步：🧮 稅額試算
- 看本年度累積到現在的預估稅額
- 情境試算：「再接一個 10 萬的案子，要多繳多少？」

## 5. 月底 workflow

1. **📤 匯入匯出** → 銀行 CSV 匯入：下載 Cathay / 玉山 / 台銀 / Wise 的交易明細 → 上傳 → 勾選入帳列 → 按「匯入選取」
2. **📤 匯入匯出** → 扣繳憑單 PDF 上傳：業主寄來的電子扣繳憑單丟上去 → Claude 自動抽欄位 → 確認無誤後存入
3. **⏰ 應收追蹤** → 看哪家業主還沒匯款 → 勾選產催款草稿（LLM 版本用 Claude 語氣幫妳寫）→ 複製貼到 email / LINE

## 6. 年度 workflow（5 月）

1. 到財政部 MyData / 綜合所得稅申報系統下載扣繳清單
2. 對照「📋 收入明細」+「扣繳憑單」是否一致
3. **📄 報稅草稿** → 一鍵下載 Markdown → 對著財政部官方軟體填

## 7. LLM 功能要填 key（可選）

扣繳憑單 OCR + 催款文字 LLM 版 需要 Anthropic API key。

```bash
export ANTHROPIC_API_KEY=sk-ant-...
# 然後再 streamlit run ui/app.py
```

沒 key 也沒關係：
- 扣繳憑單可以手動 key-in
- 催款文字會 fall back 到本地模板

## 8. 資料備份

`data/diana.db` 是 SQLite 單檔，直接複製就是備份。
預設已經 gitignore 掉，不會上雲。

## 9. 遇到問題

- 看 `PLAN.md` 的「13. 已知風險」段
- 稅法參數每年 11 月要更新 → 改 `core/rules_114.py`
- Streamlit 出錯 → 關掉重開，或 `streamlit cache clear`

## 10. 幾個她常問的坑

- **9B 稿費 vs 50 授課怎麼分？**
  - 公司員工訓練、訓練班、研習系列 = 50
  - 一次性對外演講、學術演講 = 9B
- **18 萬免稅額是每個案主 18 萬還是全年 18 萬？**
  - 全年合計。稿費+版稅+講演 全部加總 ≤ 18 萬免稅。
- **超過 20,000 才扣二代健保，20,000 要算嗎？**
  - 要。≥ 20,000 就扣。單筆看，不是全年。
- **扣繳 10% 是「預扣」，年底結算可能退？**
  - 對。5 月申報時，已扣繳金額從應納稅額扣掉，結果可能退稅。
